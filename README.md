TestVbAutomation
================

Demonstrates how to automate Rhino 5, via COM, from a Visual Basic 6.0 application.

Building Sample
--------------------
To build the sample, you are going to need:

* Rhinoceros 5 (http://www.rhino3d.com)
* Microsoft Visual Basic 6.0
